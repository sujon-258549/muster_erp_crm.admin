# 01 — Pre-Sale (বিক্রির আগে)

কোম্পানি যখন interest দেখাচ্ছে কিন্তু এখনো রাজি হয়নি — সেই stage-এর জন্য এই checklist।

---

## Step 1: কোম্পানির প্রয়োজন বুঝুন

প্রথম meeting-এ এই প্রশ্নগুলো জিজ্ঞেস করুন:

- [ ] কোম্পানির **কতটা branch** আছে? (one location / multiple / multi-city)
- [ ] **কতজন employee** আছে?
- [ ] **কী business** — Product বিক্রি, Service, নাকি দুটোই?
- [ ] এখন কোন software ব্যবহার করছে? (Tally / Excel / কিছুই না)
- [ ] **মাসে কতগুলো invoice** তৈরি হয়?
- [ ] Inventory track করতে হবে কি না?
- [ ] **Online sale** আছে কি না (blog / catalog দরকার কি না)?

> এই উত্তরগুলো থেকে বুঝবেন তাদের জন্য কোন plan suitable।

---

## Step 2: Demo দেখান

Demo দেওয়ার জন্য একটা **demo account** তৈরি করে রাখুন — প্রতি client-এর জন্য নতুন setup করবেন না।

### Demo দেখানোর ক্রম

```
1. Login screen
2. Dashboard — KPI cards
3. Branches → Sub-Branches  (একটা company multi-location কীভাবে handle করে)
4. Employee Management
   - Departments
   - Roles + Permission modal      ← এটা highlight করবেন
   - Designations
   - Employee list
5. Product / Category setup
6. Customer + Invoice creation
7. Reports / Inventory
```

### Demo-তে যা **must show**

- **Role-based permission** — কোন employee কী দেখতে পারবে, super admin control করে
- **Sticky Action column** — table-এ অনেক column থাকলেও edit/delete হারায় না
- **Active / Deactive toggle** — যেকোনো entry এক click-এ deactivate
- **Mobile responsive** — phone থেকেও দেখা যায়

---

## Step 3: Plan & Price বলুন

Subscription plans-এর জন্য root-এর `WORKFLOW.md`-এ Section 6 দেখুন। Pricing দেওয়ার আগে confirm করুন:

| Plan parameter | কী জিজ্ঞেস করবেন |
| --- | --- |
| User count | কতজন employee login নিবে? |
| Branch count | কতটা branch / sub-branch? |
| Duration | মাসিক, ৬ মাস, না বার্ষিক? |
| Storage | Media library-তে কত GB লাগবে? |
| Support level | শুধু email, না phone-ও দরকার? |

---

## Step 4: Quotation ও Agreement

- [ ] Written quotation পাঠান (PDF / email)
- [ ] Agreement / contract sign করান
- [ ] Advance payment নিন (পুরো বা partial)
- [ ] **Payment receipt** issue করুন

> ⚠️ Payment confirm হওয়ার আগে কোনোভাবেই onboarding শুরু করবেন না।

---

## Step 5: Information collection form

কোম্পানি রাজি হলে তাদের কাছ থেকে নিচের information আগেই সংগ্রহ করে রাখুন। এতে [02-onboarding-a-company.md](./02-onboarding-a-company.md) দ্রুত শেষ হবে।

```
Company name        : ______________________
Trade license       : ______________________
Address             : ______________________
Logo (file)         : ______________________

Branches:
  Main branch       : নাম + ঠিকানা
  Sub-branches      : নাম + ঠিকানা (যদি থাকে)

Departments (যেমন: Sales, HR, Accounts, IT, ...)

Designations (যেমন: Manager, Officer, Executive, ...)

Roles প্রয়োজনীয় (যেমন: Admin, Manager, Sales-Rep, Accountant)

Company Admin:
  Name              : ______________________
  Email             : ______________________
  Phone             : ______________________
```

---

## Step 6: Pre-sale checklist (final)

বিক্রি confirm করার আগে নিশ্চিত করুন:

- [ ] Demo দেওয়া হয়েছে
- [ ] Plan ঠিক হয়েছে
- [ ] Price agree হয়েছে
- [ ] Agreement signed
- [ ] Advance payment received
- [ ] Information form filled
