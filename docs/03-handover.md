# 03 — Handover (কোম্পানির হাতে তুলে দেওয়া)

Setup শেষ। এখন কোম্পানিকে handover করতে হবে। এই step skip করলে customer satisfaction কমে যাবে।

---

## Step 1: Handover meeting arrange করুন

- **মাধ্যম:** সরাসরি অফিসে গিয়ে, না হলে Zoom / Google Meet
- **সময়:** কমপক্ষে **১ ঘন্টা** রাখুন
- **কাদের রাখবেন:**
  - কোম্পানির **owner / MD** (Company Admin হবে যে)
  - তাদের IT / accounts manager (যে দৈনিক ব্যবহার করবে)

---

## Step 2: Welcome documents দিন

হাতে / email-এ এই documents পাঠান:

### 📄 ১. Welcome Letter

```
Subject: Welcome to Muster ERP — Your Account is Ready!

প্রিয় [কোম্পানির নাম],

আপনাদের Muster ERP / CRM account সম্পূর্ণ ভাবে setup
হয়ে গেছে। নিচে আপনার login তথ্য দেওয়া হলো —

URL       : https://app.muster-erp.com  (or আপনার actual URL)
Email     : [company admin email]
Password  : [strong password — আপনি set করেছেন]

প্রথম login-এর পর Settings থেকে password change
করে নিন।

Subscription valid until: [end date]

কোনো সমস্যা হলে আমাদের জানান —
  Phone: [আপনার number]
  Email: [আপনার email]

ধন্যবাদ।
```

### 📄 ২. User Manual (PDF)

কোম্পানির জন্য তৈরি করা manual — কোন button কী কাজ করে।  
*(আপনি একবার তৈরি করে সব client-কে দিতে পারেন।)*

### 📄 ৩. Invoice / Payment receipt

Payment-এর official invoice + receipt — accounts-এর জন্য।

---

## Step 3: Training session — live demo

Handover meeting-এ live demo দিয়ে নিচের জিনিসগুলো হাতে-কলমে দেখান:

### ⏱ ১৫ মিনিট — Basics

```
[ ] Login করা
[ ] Password change করা (Settings page)
[ ] Profile picture upload (যদি থাকে)
[ ] Dashboard navigate করা
[ ] Logout করা
```

### ⏱ ২০ মিনিট — Daily operations

```
[ ] নতুন Customer add করা
[ ] নতুন Product add করা
[ ] Inventory check করা
[ ] Invoice তৈরি করা ← practical example দিয়ে দেখান
[ ] Invoice print / PDF download
```

### ⏱ ১৫ মিনিট — Team management

```
[ ] নতুন Employee add করা
[ ] Role-এ permission ঠিক করা
[ ] কাউকে Deactive করা (toggle)
[ ] Employee edit করা
```

### ⏱ ১০ মিনিট — প্রশ্ন-উত্তর

তাদের যেকোনো প্রশ্ন address করুন। নোট নিয়ে রাখুন — পরের training session improve করতে কাজে দিবে।

---

## Step 4: কোম্পানিকে first task দিয়ে দিন

Meeting শেষ করার আগে তাদের একটা **practice task** দিন যাতে নিজেরা করে দেখে:

> 🎯 **Practice task:**
> "এখন নিজে একজন employee add করুন, একজন customer add করুন, এবং একটা sample invoice তৈরি করুন।"

আপনি পাশে বসে guide করুন কিন্তু **নিজে করবেন না** — তাদেরই হাতে করতে দিন।

---

## Step 5: Support channel ঠিক করে নিন

কোম্পানিকে বলুন কোন channel-এ সমস্যা জানাবে এবং কতক্ষণে response পাবে:

| Issue type | Channel | Response time |
| --- | --- | --- |
| Critical (login হচ্ছে না, data missing) | Phone | ৩০ মিনিট |
| Normal (কোনো feature বুঝতে সমস্যা) | WhatsApp / Email | ৪ ঘন্টা |
| Feature request | Email | ১-৩ দিন |

---

## Step 6: Handover sign-off

কোম্পানি থেকে একটা handover acknowledgment নিন (signed paper বা email reply):

```
আমি [নাম], [কোম্পানির নাম] এর পক্ষ থেকে নিশ্চিত করছি
যে Muster ERP / CRM সফলভাবে আমাদের হাতে handover করা
হয়েছে। Login working, training সম্পন্ন এবং আমরা
ব্যবহার শুরু করতে পারছি।

Date    : ___________
Sign    : ___________
```

এই sign-off-এর পর project officially "delivered" — পরে কোনো dispute হলে এটা proof।

---

## Handover checklist (final)

- [ ] Welcome letter দেওয়া হয়েছে (login + password সহ)
- [ ] Manual / documentation দেওয়া হয়েছে
- [ ] Live training সম্পন্ন
- [ ] Practice task করানো হয়েছে
- [ ] Support channel ঠিক হয়েছে
- [ ] Sign-off acknowledgment আছে
- [ ] কোম্পানির phone-এ আপনার number save করিয়েছেন
