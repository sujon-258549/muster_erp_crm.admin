# 04 — Post-Sale Support & Renewal

বিক্রির পর এখানেই আসল ব্যবসা — সঠিক support দিলে renewal হবে, রেফারেল আসবে।

---

## First 7 days — "Honeymoon period"

প্রথম ৭ দিন কোম্পানি সবচেয়ে বেশি প্রশ্ন করবে। এখানে support strong রাখলে customer-এর confidence দ্বিগুণ হয়।

### Daily routine

```
Day 1   :  Handover-এর পরের দিনই একটা call দিন — "সব ঠিক আছে?"
Day 2-3 :  WhatsApp-এ check-in — "কোনো সমস্যা?"
Day 5   :  ওদের system-এ login দেখুন — কতগুলো invoice হলো, কতজন employee
Day 7   :  Feedback call — "১ সপ্তাহের অভিজ্ঞতা কেমন?"
```

> এই ৭ দিনে যা শিখবেন সেটা পরের client-এর demo + onboarding-কে আরো sharp করবে।

---

## First 30 days — Adoption tracking

কোম্পানি actually system ব্যবহার করছে কি না সেটা **Super Admin login** থেকে check করুন:

### কী দেখবেন

- [ ] কতজন employee actually login করেছে?
- [ ] কতগুলো invoice generate হয়েছে?
- [ ] Customer / Product list বাড়ছে কি না?
- [ ] কোন module-গুলো **একদমই ব্যবহার হচ্ছে না**?

> ❗ যদি ১৫ দিনেও invoice না হয় — কোম্পানির সাথে call দিয়ে কারণ জানুন। হয়তো কোনো confusion আছে।

---

## Ongoing support প্রক্রিয়া

### Issue tracking

প্রতিটা সমস্যা log করে রাখুন (Excel / Notion / Trello):

```
Date | Company | Issue | Priority | Status | Resolution time
-----|---------|-------|----------|--------|----------------
```

এটা দেখলে বুঝবেন —
- কোন কোম্পানি সবচেয়ে বেশি সমস্যা করছে
- কোন feature-এ bug আছে যেটা সব client-এর
- আপনার response time কেমন

### Common সমস্যা ও solution

| সমস্যা | কী জিজ্ঞেস করবেন | Solution |
| --- | --- | --- |
| "Login হচ্ছে না" | কোন email, কী error দেখাচ্ছে | Password reset / subscription check |
| "Module দেখা যাচ্ছে না" | কোন role-এর user | Permission modal-এ গিয়ে role-এ permission দিন |
| "Invoice print হচ্ছে না" | Browser কী, কী error | Browser update / print template fix |
| "Slow হয়ে গেছে" | কখন থেকে, কোন page | Server check / data volume দেখুন |

## Subscription renewal

### Expiry এর ৩০ দিন আগে

```
[ ] কোম্পানিকে reminder mail
[ ] Renewal quotation send
[ ] Call দিয়ে confirm — "renew করছেন তো?"
```

### Expiry এর ৭ দিন আগে

```
[ ] Final reminder
[ ] Payment terms confirm
[ ] কোনো price change থাকলে আগেই জানান
```

### Expiry এর দিন

```
[ ] Payment received → subscription extend (/subscriptions)
[ ] Payment না হলে → grace period set (৩-৭ দিন)
[ ] Grace period শেষেও payment না এলে → access restrict
```

## Upgrade / Upsell সুযোগ

কোম্পানি ব্যবহার করতে করতে এই signals দিলে — upgrade pitch করুন:

| Signal | Upgrade pitch |
| --- | --- |
| ৫০+ employee হয়ে গেছে | Higher plan (more users) |
| Multiple branch খুলছে | Multi-branch plan |
| Storage প্রায় full | Storage add-on |
| Custom report চাইছে | Premium plan |
| WhatsApp / SMS integration জিজ্ঞেস করছে | Add-on module |

---

## Referral program (gold mine)

প্রতি handover-এর সময় বলুন:

> "আপনার পরিচিত কোনো কোম্পানি এই system নিতে চাইলে আমাদের জানাবেন।  
>  Reference দিলে — তাদের জন্য ১০% discount, আপনার জন্য ১ মাস free extension।"

এক customer থেকে গড়ে ১.৫ জন নতুন customer আসা সম্ভব এই formula-তে।

---

## Churn signal — যা দেখলে সতর্ক হবেন

| Signal | কী করবেন |
| --- | --- |
| ৭ দিন কেউ login করেনি | Call দিন — কী হলো |
| বারবার একই সমস্যা report করছে | Personal training session arrange করুন |
| Renewal এর কাছাকাছি কোনো response নেই | Owner-এর সাথে সরাসরি কথা বলুন |
| Competitor-এর নাম জিজ্ঞেস করছে | Retention discount offer করুন |

---

## Long-term সম্পর্ক

প্রতি ৩ মাসে একবার:

- [ ] কোম্পানির সাথে review meeting (১৫-৩০ মিনিট)
- [ ] নতুন কোনো feature add হলে দেখান
- [ ] তাদের suggestion শুনুন
- [ ] Festival-এ গ্রিটিংস পাঠান (ঈদ, পূজা, নববর্ষ)

> ভালো relationship = renewal guaranteed = stable revenue।

---

## Post-sale checklist (recurring)

প্রতি client-এর জন্য এই checklist maintain করুন:

```
Client: ____________________

[ ] Week 1 — daily check-in
[ ] Week 4 — adoption review
[ ] Month 3 — quarterly review
[ ] Month 6 — half-yearly review
[ ] 30 days before expiry — renewal reminder
[ ] 7 days before expiry — final reminder
[ ] Renewal date — payment + extension
```
